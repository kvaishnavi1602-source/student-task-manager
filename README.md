# student-task-manager
An interactive task management SPA, built to learn DOM manipulation, with Vanilla JavaScript, HTML5, and CSS3. 

This project was built to master client-side web fundamentals, specifically focusing on advanced DOM manipulation, state management, and event-driven programming without relying on external frameworks like React or Angular.

##  Features

* **Dynamic Task Management:** Add, edit, delete, and mark tasks as completed.
* **Smart Filtering & Search:** Search tasks in real-time by text, or filter them by custom categories (Personal, College Study, Others).
* **Sorting Algorithms:** Sort your to-do list by priority weight (High, Medium, Low) or chronologically by Due Date.
* **Interactive UI:** Features custom CSS variables, pastel color coding for priorities, strike-through completion states, and delete-confirmation prompts.

##  Technologies Used

* **HTML5:** Semantic structure and form elements.
* **CSS3:** Flexbox layout, CSS variables (`:root`), and responsive design.
* **Vanilla JavaScript:** * Strict DOM Manipulation (`createElement`, `appendChild`)
  * Array Methods (`filter`, `map`, `sort`)
  * Event Listeners & State Management
